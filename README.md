# CERP Video Website

这是一个教育视频内容平台，提供留学和旅行相关的视频资源。

## 功能特点

- 响应式网页设计
- 视频播放功能
- Font Awesome图标支持

## 目录结构

```
├── CERP.html                    # 主页面
├── overseas_student_safety.html # 留学生安全页面
├── css/                        # 样式文件
│   └── font-awesome/          # Font Awesome图标
└── Vodes/                     # 视频文件目录
```

## 部署说明

由于视频文件较大，我们采用以下部署策略：

1. 代码文件通过GitHub托管
2. 视频文件需要单独部署到视频存储服务器
3. HTML文件中的视频链接会指向视频存储服务器的地址

## 开发

1. 克隆仓库
2. 修改HTML/CSS文件
3. 提交并推送更改

## 注意事项

- 视频文件不包含在Git仓库中，需要单独部署到视频存储服务器
- 确保视频文件的访问权限设置正确
- 定期检查视频链接的有效性
